# Startup Business Model

Gencode is a premium technical-skill growth platform for learners, professionals, teams, bootcamps, and institutions.

## Customers

- Individual learners: students, career switchers, self-taught developers.
- Interview candidates: users preparing for backend, data, DevOps, and software engineering interviews.
- Professionals: engineers improving Linux, SQL, debugging, APIs, Git, and DSA.
- Teams: bootcamps, classrooms, coding clubs, engineering enablement, hiring cohorts.
- Enterprise: companies, colleges, and academies needing custom training and reporting.

## Revenue

- B2C subscriptions: Starter, Pro, Elite.
- B2B seats: Team plan with minimum seats.
- Enterprise contracts: custom challenge packs, private paths, analytics, support, SSO-ready architecture.

## Expansion Path

1. Free user clears first challenge.
2. User hits Genie/challenge/difficulty limit.
3. User upgrades to Starter or Pro.
4. Advanced user upgrades to Elite for Insane/VR/interview intensity.
5. User shares public profile or brings cohort/team.
6. Team lead converts to Team or Enterprise.

## Risks

- Challenge quality must remain high enough to justify subscription.
- Genie must guide without leaking answers too early.
- Runner infrastructure must become isolated and scalable before real code execution.
- Team analytics must become genuinely useful before heavy B2B sales.

## Next Business Experiments

- Founder annual discount.
- Interview-prep path bundle.
- Bootcamp pilot cohort.
- Employer-sponsored skill profile.
- Elite Insane Gauntlet launch campaign.
